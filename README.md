# Bolsa Futuro Digital - JavaScript/TypeScript - Node.js  🎓

Este repositório contém todo o material prático, exercícios e projetos desenvolvidos durante o programa **Bolsa Futuro Digital**. O foco do aprendizado abrange desde lógica de programação pura com JavaScript até o desenvolvimento de APIs robustas com TypeScript, Node.js e MySQL.

## 🚀 Tecnologias Utilizadas

* **Linguagens:** JavaScript (ES6+), TypeScript
* **Runtime:** Node.js
* **Framework Web:** Express.js
* **Banco de Dados:** MySQL
* **Ferramentas:** Nodemon, Ts-node-dev, Jest (Testes)

---

## 📂 Estrutura do Repositório

### 1. Banco de Dados (`/MySQL`)
Scripts SQL estruturados para criação de bancos, tabelas e consultas complexas (Joins, Agrupamentos, Subqueries).
* **Escola TI**: Sistema de gestão de alunos e matrículas.
* **Hardware Store**: Cadastro de produtos de alta performance (GPUs RTX Series).

### 2. Fundamentos e Lógica (`/pratica/aula1` a `aula12`)
Exercícios iniciais focados em:
* Operadores aritméticos e lógica condicional (`if/else`, `switch`).
* Estruturas de repetição (`for`, `while`, `forEach`).
* Manipulação de Arrays e Objetos.
* Introdução à Programação Orientada a Objetos (POO) em JS.

### 3. TypeScript e POO Avançada (`/pratica/aula13` a `aula18`)
Transição para TypeScript, aplicando tipagem estática e conceitos de:
* **Classes e Entidades**: Sistemas de Barbearia, Livraria e Bancos.
* **Relacionamentos**: Implementação de associações 1:N e N:N entre objetos (ex: Autor x Livro).
* **Modulação**: Separação de responsabilidades em arquivos distintos.

### 4. Desenvolvimento de APIs (`/pratica/aula28-30(API)`)
Construção de uma API RESTful completa utilizando o padrão de arquitetura em camadas:
* **Controller**: Gerenciamento das requisições e respostas.
* **Service**: Regras de negócio e validações.
* **Repository**: Interface de comunicação com o banco de dados MySQL.
* **Router**: Definição dos endpoints (GET, POST, PUT, DELETE).
